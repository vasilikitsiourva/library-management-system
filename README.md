# Library Management System

## Overview
This project is a library management system implemented in C, developed as part of a university Data Structures course.
It simulates the core functionality of a library including managing books users and borrowing operations.

The focus of the project is on applying fundamental data structures and designing an efficient system for handling structured data.



## Features
- Add, remove, and manage books
- Manage user records
- Borrow and return functionality
- Basic data handling and storage logic
- Command-line based interaction
